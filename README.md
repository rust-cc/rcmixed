# rcmixed
Mixed cryptosystem. inspired by PGP.

- [x] key store.
- [x] public key cryptography + symmetric cryptography.
- [ ] DH cryptography + symmetric cryptography.
- [ ] feature implement:
  - [ ] RSA + AES-CBC + SHA(2, 3)
  - [ ] DH-RSA + AES-CBC + SHA(2, 3)
  - [ ] x25519(DH + curve25519) + AES-CBC + SHA(2, 3)
- [ ] Failure.
- [ ] Tests.
- [ ] Bench.
- [ ] Doc.
